div etiketi (division kısaltması) html etiketlerini çevreleyen, css'ler ile birlikte kullanılarak görsel tasarımlar oluşturulmasına yardımcı olan etikettir. Arayüzde görsel bir çıktı vermez.

Dünya Naber

id attribute : tüm elementlere id özelliği eklenebilir. Bir id değeri sayfa içinde sadece bir kez tanımlanabilir. (tekil unique olması gerekir.)
Class attribute: tüm elementlere class özelliği eklenebilir. Ancak id gibi tekil olma zorunlulupu yoktur birden fazla elemente aynı class adı tanımlanabilir.

Style etiketi içinde Css kodları yazarak tasarımda görsel düzenlemeler yapabiliriz. Style tag head tag'i içinde tanımlanır. Css kodlarını elementin içine de yazılabilir.
